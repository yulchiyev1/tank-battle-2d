Transform2\+D\+::\+Transform2D (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [inline]}}



Constructs a default 2D transform. 

Initializes the transform with\+:
\begin{DoxyItemize}
\item position = (0, 0)
\item rotation = 0
\item scale = (1, 1)
\item matrix = identity
\item is\+Changed = true
\item depth = 0
\end{DoxyItemize}

The matrix is not rebuilt immediately here. Instead, it is marked dirty and lazily recalculated the next time \doxylink{class_transform2_d_a86e9a6ffac2c382a9e3467f8331bf86d}{Get\+Matrix()} is called.

\begin{DoxyNote}{Note}
The depth value is stored separately from the transform matrix and is used for render ordering rather than geometric transformation. 
\end{DoxyNote}

glm\+::mat4 \& Transform2\+D\+::\+Get\+Matrix (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [nodiscard]}}



Returns the cached transform matrix. 

Rebuilds the matrix only when the transform has changed. In the current implementation, the matrix is composed as\+:

position translation \texorpdfstring{$\ast$}{*} offset translation \texorpdfstring{$\ast$}{*} rotation \texorpdfstring{$\ast$}{*} scale

After rebuilding, the dirty flag is cleared and the cached matrix is reused until another transform-\/changing function marks it dirty again.

\begin{DoxyReturn}{Returns}
Reference to the internally cached 4x4 transform matrix.
\end{DoxyReturn}
\begin{DoxyNote}{Note}
The returned reference remains valid as long as this \doxylink{class_transform2_d}{Transform2D} instance exists. The depth value is not included in this matrix. 
\end{DoxyNote}
