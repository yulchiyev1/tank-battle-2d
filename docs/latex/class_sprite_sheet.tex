\doxysubsection{Detailed Description}
Represents a grid-\/based sprite sheet and stores named animation clips. 

\doxylink{class_sprite_sheet}{Sprite\+Sheet} interprets a texture as a regular grid of frames of equal size. The constructor calculates the number of columns and rows from the texture dimensions and the frame dimensions. If the calculated values become zero, the implementation clamps them to at least one so that frame lookup remains valid.

This class is responsible for\+:
\begin{DoxyItemize}
\item converting frame indices into UV offsets,
\item returning normalized UV scale for one frame,
\item storing named \doxylink{struct_sprite_clip}{Sprite\+Clip} entries for clip-\/based animation playback.
\end{DoxyItemize}

The UV calculation flips the row index vertically so that sprite frame lookup matches the engine\textquotesingle{}s texture coordinate convention. This behavior is implemented in \doxylink{class_sprite_sheet_a2052999ec8fce40951d4df88118d908a}{Get\+UVOffset()} rather than being handled externally. \+:content\+Reference\mbox{[}oaicite\+:1\mbox{]}\{index=1\}

\begin{DoxyNote}{Note}
This class does not manage playback time or frame advancement. It only provides frame layout data and clip lookup. 
\end{DoxyNote}

Returns the UV offset of the specified frame. 

Converts a zero-\/based frame index into a column and row position inside the sprite sheet grid. The row is then vertically flipped before converting the result into normalized UV coordinates.

The returned value represents the UV origin of the frame region. The per-\/frame UV size is provided separately by \doxylink{class_sprite_sheet_a5ff75353185084b8624ee771d10a590a}{Get\+UVScale()}.
