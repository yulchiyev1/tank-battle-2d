\doxysubsection{Detailed Description}
Controls sprite animation playback using a \doxylink{class_sprite_sheet}{Sprite\+Sheet}. 

\doxylink{class_sprite_animator}{Sprite\+Animator} supports two playback modes\+:
\begin{DoxyEnumerate}
\item raw frame range playback using \doxylink{class_sprite_animator_af2935db4720fc2387d34796559412404}{Play\+Clip(int start, int end, bool loop\+\_\+)},
\item named clip playback using \doxylink{class_sprite_animator_aac8752dcb1e73427d97e4f3126ec841a}{Play\+Clip(const std\+::string\& clip\+Name)}.
\end{DoxyEnumerate}

During \doxylink{class_sprite_animator_a23f9880b2fdbbd23fe48f25526cafb07}{Update()}, the animator advances elapsed time and changes frames when the accumulated time reaches the current frame duration. For named clips, the duration comes from \doxylink{struct_sprite_clip_a523b8e6b2ae91841db8d0a695af55728}{Sprite\+Clip\+::frame\+Duration}. For raw frame range playback, the duration comes from frame\+Time.

This class also supports\+:
\begin{DoxyItemize}
\item global playback speed control,
\item per-\/clip playback speed overrides,
\item frame callbacks by absolute frame index,
\item frame callbacks by clip-\/local frame index,
\item one-\/shot callbacks that remove themselves after execution.
\end{DoxyItemize}

A key behavior of this implementation is that \doxylink{class_sprite_animator_af2935db4720fc2387d34796559412404}{Play\+Clip()} immediately triggers callbacks for the current starting frame instead of waiting for the next \doxylink{class_sprite_animator_a23f9880b2fdbbd23fe48f25526cafb07}{Update()}. This means callbacks registered for the initial frame can fire as soon as playback begins. \+:content\+Reference\mbox{[}oaicite\+:5\mbox{]}\{index=5\}

\begin{DoxyNote}{Note}
The animator does not own the texture directly. It queries the current \doxylink{class_sprite_sheet}{Sprite\+Sheet} when UV scale, UV offset, or texture access is needed. 
\end{DoxyNote}

Advances animation playback using delta time. 

Accumulates elapsed time and advances frames whenever the active frame duration has been exceeded.

Behavior differs depending on playback mode\+:
\begin{DoxyItemize}
\item If a named clip is active, \doxylink{class_sprite_animator_a23f9880b2fdbbd23fe48f25526cafb07}{Update()} advances clip\+Frame\+Index through the clip\textquotesingle{}s frame\+Indices vector using the clip\textquotesingle{}s frame\+Duration.
\item If no named clip is active, \doxylink{class_sprite_animator_a23f9880b2fdbbd23fe48f25526cafb07}{Update()} advances current\+Frame from start\+Frame to end\+Frame using the animator\textquotesingle{}s default frame\+Time.
\end{DoxyItemize}

Playback speed is affected by\+:
\begin{DoxyItemize}
\item the global playback\+Speed,
\item the per-\/clip playback speed override if a named clip is active.
\end{DoxyItemize}

When playback reaches the end\+:
\begin{DoxyItemize}
\item looping playback wraps to the first frame,
\item non-\/looping playback stays on the last frame and sets is\+Clip\+Finished to true.
\end{DoxyItemize}

Whenever a new current frame becomes active, the animator triggers the relevant callbacks for that frame. \+:content\+Reference\mbox{[}oaicite\+:9\mbox{]}\{index=9\}
