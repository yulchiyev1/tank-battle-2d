Constructs a \doxylink{class_window_manager}{Window\+Manager} with default window state. 

Initializes the internal window handle to nullptr and sets the default window size to 800x600. The default clear color is initialized to (0.\+4, 0.\+4, 0.\+4, 1.\+0). This constructor only prepares the manager state; it does not create a GLFW window or initialize Open\+GL.

\begin{DoxyNote}{Note}
Actual window creation happens later through \doxylink{class_window_manager_a5b4ee360c72b19e135eeac7267074964}{Init()}. 
\end{DoxyNote}

Registers or replaces a resize callback under a string key. 

Stores the callback in an internal unordered\+\_\+map using the given key. If the same key already exists, the previous callback is overwritten. Registered callbacks are invoked from \doxylink{class_window_manager_ac494c42b664babdcd3871974eae6fb52}{Notify\+Resize()}, which is triggered by the framebuffer size callback after engine-\/side resize handling has already updated the window size, active state camera sizes, and render target sizes.

Clears the framebuffer, scroll, key, mouse button, and close callbacks, removes the user pointer, and destroys the GLFW window. This is part of the engine shutdown sequence executed after state, render, and sound systems have been freed.

Creates the GLFW window and initializes the window subsystem. 

This is the core startup routine used by \doxylink{class_jin_engine_af2b70669383ccdbbc05309c7e4eb50db}{Jin\+Engine\+::\+Init()}. It initializes GLFW, configures the Open\+GL context version and profile, creates the native window, makes the context current, loads GLAD, enables Open\+GL debug output, sets the initial viewport, stores the engine pointer as the GLFW user pointer, and installs the engine\textquotesingle{}s window/input callbacks.

Installed callbacks include\+:
\begin{DoxyItemize}
\item framebuffer size callback for resize propagation
\item scroll callback for \doxylink{class_input_manager}{Input\+Manager} scroll accumulation
\item key callback for staged keyboard input
\item mouse button callback for staged mouse input
\item window close callback to request engine shutdown
\end{DoxyItemize}

void Window\+Manager\+::\+Set\+Full\+Screen (\begin{DoxyParamCaption}\item[{bool}]{enable}{}\end{DoxyParamCaption})}



Switches the window between windowed mode and fullscreen mode. 

When entering fullscreen, the current windowed position and size are saved so that they can be restored later. The function then attaches the window to the primary monitor using that monitor\textquotesingle{}s current video mode. When leaving fullscreen, the saved windowed position and size are restored.

The internal cached width and height are updated immediately after the mode switch. This function returns early if no window exists or if the requested fullscreen state already matches the current state.


\begin{DoxyParams}{Parameters}
{\em enable} & True to switch to fullscreen mode, false to restore windowed mode.\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
This function updates the cached size values directly, but any framebuffer-\/dependent systems should still rely on the actual resize callback path for full synchronization. 
\end{DoxyNote}
