\doxysubsection{Detailed Description}
Central input state manager for keyboard, mouse buttons, cursor position, and scroll input. 

\doxylink{class_input_manager}{Input\+Manager} collects raw input events from GLFW callbacks and converts them into frame-\/stable queryable states for gameplay and engine systems.

The current implementation uses a staged input model\+:
\begin{DoxyItemize}
\item GLFW callbacks write into staged\+Key\+State and staged\+Mouse\+State through \doxylink{class_input_manager_ae569ff5be56bca62b47a7bdfffe2c867}{On\+Key()} and \doxylink{class_input_manager_a84f610e644a2de5cb64b55220d4a4c8e}{On\+Mouse\+Button()}.
\item \doxylink{class_input_manager_aa5480931dba2720e7d80dd00a53adae0}{Update()} copies staged states into current states once per frame.
\item previous states are preserved so pressed/released transitions can be queried.
\end{DoxyItemize}

This allows user code to call\+:
\begin{DoxyItemize}
\item \doxylink{class_input_manager_afd4bcb5fd69d6262fa79f64a4ae43b01}{Is\+Key\+Down()}
\item \doxylink{class_input_manager_ac052ac0d0831d40848031aca922a00ef}{Is\+Key\+Pressed()}
\item \doxylink{class_input_manager_ab0e1cd02dfbeea18402babfa3a2cc88e}{Is\+Key\+Released()}
\item \doxylink{class_input_manager_adb20eaed0386dab5239202ea0022e313}{Is\+Mouse\+Button\+Down()}
\item \doxylink{class_input_manager_a9d4212a5b4b4485a9e5934930ab4de2a}{Is\+Mouse\+Button\+Pressed()}
\item \doxylink{class_input_manager_a1adf3a1c58a545d0721479bbbedf81c3}{Is\+Mouse\+Button\+Released()}
\item \doxylink{class_input_manager_afb908706c127374a1450c3a6400677f1}{Is\+Mouse\+Button\+Dragging()}
\end{DoxyItemize}

Cursor position is sampled every frame from GLFW, and scroll input is accumulated asynchronously through \doxylink{class_input_manager_a9a7e6c988123d556523b698ae010cc5b}{Add\+Scroll()} before being exposed as per-\/frame deltas in \doxylink{class_input_manager_aa5480931dba2720e7d80dd00a53adae0}{Update()}.

\doxylink{class_input_manager}{Input\+Manager} is initialized and driven by \doxylink{class_jin_engine}{Jin\+Engine}\+:
\begin{DoxyItemize}
\item \doxylink{class_jin_engine}{Jin\+Engine} calls \doxylink{class_input_manager_a222eea359d455ca0517bde4f74536754}{Init()} with the GLFW window handle.
\item \doxylink{class_jin_engine}{Jin\+Engine} calls \doxylink{class_input_manager_aa5480931dba2720e7d80dd00a53adae0}{Update()} once per frame before game-\/state update.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
This class stores input state only. It does not own the window.

World-\/space mouse queries require a valid \doxylink{class_camera2_d}{Camera2D} pointer. 
\end{DoxyNote}
