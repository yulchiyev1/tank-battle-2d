\doxysubsection{Detailed Description}
Owns GPU-\/side mesh buffers and provides draw operations for normal and instanced rendering. 

\doxylink{class_mesh}{Mesh} is the engine\textquotesingle{}s low-\/level geometry container. It creates and owns the Open\+GL objects required to render a piece of geometry\+:
\begin{DoxyItemize}
\item a standard VAO for regular drawing
\item a VBO for vertex data
\item an optional EBO for indexed rendering
\item a dedicated instancing VAO and per-\/instance buffers for batched rendering
\end{DoxyItemize}

During construction, the mesh uploads the provided vertex/index data to GPU buffers and computes local bounds from the source vertices. Those bounds are later used by \doxylink{class_object_ac5adc371bfc7b4216ffbc85907830e25}{Object\+::\+Get\+Bounding\+Radius()} to estimate visibility and culling size.

Instanced rendering support is not automatically enabled just because this class owns instance buffers. The mesh must first be prepared through \doxylink{class_material_aea9bb91a6549dadd72c68d17100dc388}{Material\+::\+Enable\+Instancing()}, which internally calls \doxylink{class_mesh_a66fa02b2a66182031bb00a9aec134526}{Setup\+Instance\+Attributes()} after verifying that the bound shader supports instancing attributes. \+:content\+Reference\mbox{[}oaicite\+:5\mbox{]}\{index=5\}

At runtime\+:
\begin{DoxyItemize}
\item regular rendering uses \doxylink{class_mesh_abf84fe4ef735490769268685087e0e32}{Draw()}
\item batched rendering uploads per-\/instance data through \doxylink{class_mesh_a45673d65011bf8f70c0f7de2a37d7f6c}{Update\+Instance\+Buffer()} and then calls \doxylink{class_mesh_ae2933e498cc249ab798bbf265dace77b}{Draw\+Instanced()}
\end{DoxyItemize}

The \doxylink{class_render_manager}{Render\+Manager} uses these paths when flushing draw commands. \+:content\+Reference\mbox{[}oaicite\+:6\mbox{]}\{index=6\}

\begin{DoxyNote}{Note}
This class is intentionally tightly integrated with \doxylink{class_material}{Material} and \doxylink{class_render_manager}{Render\+Manager}. Internal GPU setup and draw functions are kept private and are exposed only through friendship. 
\end{DoxyNote}

Creates a mesh from vertex data, optional index data, and a primitive type. 

This constructor initializes all Open\+GL object handles to safe default values, stores the selected primitive topology, uploads the provided geometry to GPU buffers by calling \doxylink{class_mesh_a6f0136c650ed60f43587e3d8d0c121aa}{Setup\+Mesh()}, and computes the local bounding half-\/size by calling \doxylink{class_mesh_ad097772f26af95c8f7070cbb0e8cbfef}{Compute\+Local\+Bounds()}.

If the index array is empty, the mesh is treated as a non-\/indexed mesh and will later be rendered with gl\+Draw\+Arrays-\/style calls. Otherwise, it will use indexed drawing with gl\+Draw\+Elements-\/style calls. \+:content\+Reference\mbox{[}oaicite\+:7\mbox{]}\{index=7\}

The computed local bound is later used indirectly for approximate culling and bounding radius calculation through \doxylink{class_object_ac5adc371bfc7b4216ffbc85907830e25}{Object\+::\+Get\+Bounding\+Radius()}. \+:content\+Reference\mbox{[}oaicite\+:8\mbox{]}\{index=8\}

Computes a simple local-\/space bounding half-\/size from the input vertices. 

This helper scans the vertex positions, finds the local minimum and maximum XY extents, computes the size difference, and stores half of that size in local\+Half\+Size.

The value is later used by \doxylink{class_object_ac5adc371bfc7b4216ffbc85907830e25}{Object\+::\+Get\+Bounding\+Radius()} as part of the engine\textquotesingle{}s approximate visibility and spatial checks.

Special handling\+:
\begin{DoxyItemize}
\item if the mesh has no vertices, the bound defaults to (0.\+5, 0.\+5)
\item if the mesh has exactly one vertex, a tiny non-\/zero bound is used
\end{DoxyItemize}
