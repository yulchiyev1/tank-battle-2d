\doxysubsection{Detailed Description}
Base abstract object class for all engine-\/managed runtime objects. 

\doxylink{class_object}{Object} is the common polymorphic foundation used by the engine to manage scene entities such as gameplay objects and text objects.

This class stores and exposes the core shared state needed by multiple systems\+:
\begin{DoxyItemize}
\item lifecycle state (alive / visible)
\item transform data
\item material and mesh references
\item render layer tag
\item tint color
\item optional sprite animation
\item optional collider
\item collision category / mask bits
\item camera-\/ignore behavior for HUD or screen-\/space style objects
\item UV flip settings for mirrored rendering
\end{DoxyItemize}

The renderer uses this class heavily during culling and draw submission. For example\+:
\begin{DoxyItemize}
\item \doxylink{class_frustum_culler}{Frustum\+Culler} checks \doxylink{class_object_af92e9a809354c281d464011a1f009d02}{Is\+Alive()}, \doxylink{class_object_a77092a4fc48339007ed32e7d48f6140a}{Is\+Visible()}, \doxylink{class_object_afe474c804f6bab83b89958356e1857af}{Should\+Ignore\+Camera()}, \doxylink{class_object_a4c708099448203abe3182c8b0ae340ce}{Get\+World\+Position()}, and \doxylink{class_object_ac5adc371bfc7b4216ffbc85907830e25}{Get\+Bounding\+Radius()}
\item \doxylink{class_render_manager}{Render\+Manager} reads material, mesh, animation, color, transform, layer tag, and UV flip state when building render batches and issuing draws
\end{DoxyItemize}

Derived classes must implement the full lifecycle interface and may override world-\/space queries when their positioning logic differs from the default transform-\/based behavior.

\begin{DoxyNote}{Note}
This class is abstract and cannot be instantiated directly. 
\end{DoxyNote}
