\doxysubsection{Detailed Description}
Hash specialization for \doxylink{struct_instance_batch_key}{Instance\+Batch\+Key}. 

This specialization enables \doxylink{struct_instance_batch_key}{Instance\+Batch\+Key} to be used as a key in std\+::unordered\+\_\+map and other unordered associative containers.

The current implementation hashes the three raw pointer fields separately and combines them using xor and bit shifting.

\begin{DoxyNote}{Note}
As with \doxylink{struct_instance_batch_key}{Instance\+Batch\+Key} itself, hashing is based on pointer identity, not on the pointed-\/to object contents. 
\end{DoxyNote}
