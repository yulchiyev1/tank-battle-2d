\doxysubsection{Detailed Description}
Circle-\/shaped collider implementation. 

\doxylink{class_circle_collider}{Circle\+Collider} stores both a base radius and a scaled radius. When transform scaling is enabled, \doxylink{class_circle_collider_a5679b18c3379fb6303c1f41288a4a5b7}{Get\+Radius()} derives the effective size from the owner\textquotesingle{}s world scale using the larger absolute axis. Otherwise, the manually stored scaled\+Radius value is used. \+:content\+Reference\mbox{[}oaicite\+:5\mbox{]}\{index=5\}

Circle-\/vs-\/circle checks are resolved by comparing squared distance against the squared sum of radii. Circle-\/vs-\/\+AABB checks are forwarded to \doxylink{class_a_a_b_b_collider}{AABBCollider} for mixed-\/shape resolution.
