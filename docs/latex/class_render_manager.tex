\doxysubsection{Detailed Description}
Central rendering system responsible for resource registration, batching, culling integration, and frame submission. 

\doxylink{class_render_manager}{Render\+Manager} is one of the engine\textquotesingle{}s core systems. It owns and manages shared render resources such as\+:
\begin{DoxyItemize}
\item shaders
\item textures
\item meshes
\item materials
\item fonts
\item sprite sheets
\item render layers
\end{DoxyItemize}

It also manages the runtime rendering flow for each frame\+:
\begin{DoxyItemize}
\item frame target setup through \doxylink{class_render_manager_a375da2d8afa977405cd3200f643b3e84}{Begin\+Frame()}
\item visible-\/object collection through \doxylink{class_render_manager_add99380d257141db9b2e939530311d2e}{Submit()}
\item render grouping through \doxylink{class_render_manager_a8e033cb685d2619f9b5dfec8a165e387}{Build\+Render\+Map()}
\item actual draw execution through \doxylink{class_render_manager_a620231f7f4119dbf26ef416478578cde}{Flush\+Draw\+Commands()}
\item offscreen presentation through \doxylink{class_render_manager_a61a2ee6952f8da0aee3ac21cd8033d0b}{End\+Frame()}
