\doxysubsection{Detailed Description}
Represents a renderable text object whose mesh is generated from a \doxylink{class_font}{Font} at runtime. 

\doxylink{class_text_object}{Text\+Object} is a specialized \doxylink{class_object}{Object} implementation for rendering text in the world or in screen-\/aligned space. Unlike a regular sprite/object pipeline, this class does not allow arbitrary mesh or material replacement through the usual \doxylink{class_object}{Object} interface.

Instead, \doxylink{class_text_object}{Text\+Object} builds and owns its render mesh from\+:
\begin{DoxyItemize}
\item the current \doxylink{class_font}{Font},
\item the current text string,
\item the current horizontal and vertical alignment settings.
\end{DoxyItemize}

The constructor immediately generates an initial text mesh through \doxylink{class_text_object_a0705bf6f2f40788da1c9fb60368c183b}{Update\+Mesh()}. Later, the mesh is regenerated whenever the text content, alignment, or font changes.

In addition, \doxylink{class_text_object}{Text\+Object} supports automatic mesh refresh when the backing font atlas changes. This is handled through \doxylink{class_text_object_a02f4f3ea69e456a0353031edd8bb9438}{Check\+Font\+Atlas\+And\+Mesh\+Update()}, which compares a tracked atlas version against the current font atlas version and rebuilds the mesh only when necessary.

\doxylink{class_object_manager}{Object\+Manager} explicitly calls \doxylink{class_text_object_a02f4f3ea69e456a0353031edd8bb9438}{Check\+Font\+Atlas\+And\+Mesh\+Update()} during update and when pending objects are first initialized, so \doxylink{class_text_object}{Text\+Object} can react to lazy glyph baking or atlas growth without requiring manual refresh from user code. \+:content\+Reference\mbox{[}oaicite\+:5\mbox{]}\{index=5\}

The world position and world scale returned by this class are not simple transform passthroughs. They incorporate\+:
\begin{DoxyItemize}
\item the current text size reported by the font,
\item alignment offsets,
\item optional camera-\/ignore behavior using the reference camera.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
\doxylink{class_text_object_a640e4ef674fb2192f5d29b0f81f95107}{Init()}, \doxylink{class_text_object_ac135df5aeb1986940a538ab9c1b8e10f}{Late\+Init()}, \doxylink{class_text_object_aff767a41326a8242063743e15bfb28d7}{Update()}, \doxylink{class_text_object_a534ee1073bdc4b0b2d953dfbe8166f03}{Draw()}, \doxylink{class_text_object_a7aea94fb38f3c3e3771af8f88e4a7fc4}{Free()}, and \doxylink{class_text_object_a22c3477a5f4774b57cdd641120d6a248}{Late\+Free()} are currently empty in the present implementation.

\doxylink{class_material}{Material}, mesh, and sprite animation reassignment APIs are intentionally deleted so that \doxylink{class_text_object}{Text\+Object} always stays consistent with the font-\/driven text rendering pipeline. 
\end{DoxyNote}

This function is designed for dynamic/lazy font atlas systems. It safely locks the current font and\+:


\begin{DoxyItemize}
\item if the font has expired\+:
\begin{DoxyItemize}
\item clears the inherited mesh reference,
\item clears the owned text mesh,
\item resets the tracked atlas version to 0,
\item returns immediately.
\end{DoxyItemize}
\item if the tracked atlas version matches the font\textquotesingle{}s current atlas version\+:
\begin{DoxyItemize}
\item returns without rebuilding the mesh.
\end{DoxyItemize}
\item otherwise\+:
\begin{DoxyItemize}
\item updates text\+Atlas\+Version\+Tracker,
\item regenerates the text mesh from the font,
\item stores the new mesh in both the inherited mesh handle and the owned text\+Mesh.
\end{DoxyItemize}

This function first locks the current font. If the font is unavailable, it returns a zero vector.

void Text\+Object\+::\+Update\+Mesh (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [protected]}}



Regenerates the text mesh immediately from the current font, text, and alignment. 

This function safely locks the current font and\+:


\begin{DoxyItemize}
\item if the font has expired\+:
\begin{DoxyItemize}
\item clears the inherited mesh reference,
\item clears the owned text mesh,
\item resets the tracked atlas version to 0,
\item returns immediately.
\end{DoxyItemize}
\item otherwise\+:
\begin{DoxyItemize}
\item generates a new text mesh from the font,
\item stores it in both the inherited mesh handle and the owned text\+Mesh,
\item updates text\+Atlas\+Version\+Tracker to the font\textquotesingle{}s current atlas version.
\end{DoxyItemize}
\end{DoxyItemize}

Unlike \doxylink{class_text_object_a02f4f3ea69e456a0353031edd8bb9438}{Check\+Font\+Atlas\+And\+Mesh\+Update()}, this function does not compare versions first. It always rebuilds when called and is used after explicit content/alignment/font changes.
