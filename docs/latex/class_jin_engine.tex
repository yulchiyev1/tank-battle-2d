\doxysubsection{Detailed Description}
Core engine class that manages initialization, main loop execution, and system lifecycle. 

\doxylink{class_jin_engine}{Jin\+Engine} is the central coordinator of all engine subsystems including windowing, input, rendering, sound, and state management. It owns all major managers and provides a unified \doxylink{struct_engine_context}{Engine\+Context} that is passed to all systems and game states.

The engine lifecycle is as follows\+:
\begin{DoxyEnumerate}
\item \doxylink{class_jin_engine_af2b70669383ccdbbc05309c7e4eb50db}{Init()} ˇć Initializes all subsystems and sets up \doxylink{struct_engine_context}{Engine\+Context}
\item \doxylink{class_jin_engine_ad8355b55406f0be0b2f39b910ff7913e}{Run()} ˇć Executes the main game loop
\item \doxylink{class_jin_engine_a0fff451b96b00ef74a4a9975239adfdd}{Request\+Quit()} ˇć Signals the loop to terminate
\item \doxylink{class_jin_engine_a00fa0799a110bb78192a6a643f55d309}{Free()} ˇć Cleans up resources and terminates GLFW
\end{DoxyEnumerate}

Internally, \doxylink{class_jin_engine_ad8355b55406f0be0b2f39b910ff7913e}{Run()} performs\+:
\begin{DoxyItemize}
\item Frame timing (\doxylink{struct_engine_timer}{Engine\+Timer})
\item Input processing
\item State update and rendering
\item Sound update
\item Buffer swapping
\end{DoxyItemize}

\begin{DoxyNote}{Note}
This class should typically be instantiated once per application. 
\end{DoxyNote}

void Jin\+Engine\+::\+Run (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Starts the main game loop. 

This function runs the core loop until\+:
\begin{DoxyItemize}
\item \doxylink{class_jin_engine_a0fff451b96b00ef74a4a9975239adfdd}{Request\+Quit()} is called, or
\item The window is closed by the user.
\end{DoxyItemize}

Each frame performs\+:
\begin{DoxyItemize}
\item Delta time calculation (\doxylink{struct_engine_timer}{Engine\+Timer})
\item Event polling and input update
\item \doxylink{class_state_manager}{State\+Manager} update and draw
\item \doxylink{class_sound_manager}{Sound\+Manager} update
\item Buffer swap
\end{DoxyItemize}

After exiting the loop, all subsystems are properly freed.

\begin{DoxyNote}{Note}
This function blocks until the engine terminates. 
\end{DoxyNote}
