\doxysubsection{Detailed Description}
Free\+Type-\/backed font resource that manages glyph baking, atlas storage, and text mesh generation. 

\doxylink{class_font}{Font} is responsible for\+:
\begin{DoxyItemize}
\item loading a TTF face through Free\+Type
\item creating and maintaining a single glyph atlas texture
\item lazily baking glyphs on demand
\item providing a text rendering material bound to the atlas
\item measuring UTF-\/8 text
\item generating a \doxylink{class_mesh}{Mesh} for aligned text rendering
\end{DoxyItemize}

The current implementation uses a dynamically growing single atlas. Glyphs are not pre-\/generated for all characters. Instead, characters are baked as needed through \doxylink{class_font_aa3f349ce0e57bfabb0ef4cea1431a1d6}{Try\+Bake\+Glyph()}, and the atlas is expanded when there is not enough space. \+:content\+Reference\mbox{[}oaicite\+:5\mbox{]}\{index=5\}

The material created by this class uses the internal engine text shader registered by \doxylink{class_render_manager}{Render\+Manager} during initialization, and binds the atlas texture to the uniform "{}u\+\_\+\+Font\+Texture"{}.

\begin{DoxyNote}{Note}
This class owns the Free\+Type library handle and font face and releases them in the destructor.

atlas\+Version is incremented whenever the atlas is rebuilt during expansion. External systems such as \doxylink{class_text_object}{Text\+Object} can use this value to detect when text meshes need to be regenerated. 
\end{DoxyNote}
