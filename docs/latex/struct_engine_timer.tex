\doxysubsection{Detailed Description}
Frame timing utility used for delta time calculation and FPS measurement. 

\doxylink{struct_engine_timer}{Engine\+Timer} is responsible for tracking frame time progression and computing per-\/frame delta time (dt) as well as frames per second (FPS).

It is typically used inside the main engine loop\+:
\begin{DoxyItemize}
\item \doxylink{struct_engine_timer_ad25e0e9f721178342c6f3c7b953befd9}{Start()} initializes the timer.
\item \doxylink{struct_engine_timer_abe3aa7aae238367ee3d3cee3ce7678de}{Tick()} is called every frame to compute delta time.
\item \doxylink{struct_engine_timer_aaf262d9972999ca9e988923b79c7c9eb}{Should\+Update\+FPS()} accumulates time and frame count to periodically update FPS.
\end{DoxyItemize}

The timer relies on the window system\textquotesingle{}s time source (e.\+g., glfw\+Get\+Time) accessed through \doxylink{struct_engine_context}{Engine\+Context}.

\begin{DoxyNote}{Note}
This struct does not own any system and depends on \doxylink{struct_engine_context}{Engine\+Context} for time queries. 
\end{DoxyNote}

Determines whether FPS should be updated and computes it. 

Accumulates elapsed time (fps\+Timer) and frame count (frame\+Count). When the accumulated time exceeds a threshold (commonly \texorpdfstring{$\sim$}{\string~}1 second), FPS is calculated as\+: \begin{DoxyVerb}FPS = frameCount / fpsTimer
\end{DoxyVerb}


After computation, both counters are reset.

Advances the timer by one frame and computes delta time. 

Retrieves the current time from the engine (via \doxylink{struct_engine_context}{Engine\+Context}), computes the difference from the previous frame (last\+Time), and updates last\+Time with the current value.

This function is expected to be called once per frame.
