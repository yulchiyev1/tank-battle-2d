\doxysubsection{Detailed Description}
Specialized material type used for compute shader dispatch. 

\doxylink{class_compute_material}{Compute\+Material} extends \doxylink{class_material}{Material} to support image bindings used by compute shaders. Instead of relying on ordinary sampler texture bindings for output images, it stores named image units with access qualifiers, image formats, and mip levels.

The current implementation\+:
\begin{DoxyItemize}
\item overrides \doxylink{class_compute_material_ad5a9a88431fa1b207fb59f4d52528fe2}{Is\+Compute()} to return true
\item deletes \doxylink{class_compute_material_ad7fd968bb667d1ad770605f7d816d3aa}{Set\+Texture()} because regular sampler bindings are not the intended path here
\item deletes \doxylink{class_compute_material_a8db7010b3faca6fb58e1ec219aaf3195}{Enable\+Instancing()} because compute materials are never used for instanced mesh drawing
\item overrides \doxylink{class_compute_material_a3c8944e037369072b1685f9e907bc7dc}{Send\+Data()} to bind images through \doxylink{class_texture_a28b89a5f6f78de411c85fc83c04963b9}{Texture\+::\+Bind\+As\+Image()} and then send uniforms
\end{DoxyItemize}

\doxylink{class_render_manager_ae6719dd9b794d26cf923b27f183996dd}{Render\+Manager\+::\+Dispatch\+Compute()} expects a \doxylink{class_compute_material}{Compute\+Material} with a valid destination texture and uses \doxylink{class_compute_material_af9b7118ccbf4f357223fe343bfaa562f}{Get\+Dst\+Texture()} to derive the dispatch group count. \+:content\+Reference\mbox{[}oaicite\+:15\mbox{]}\{index=15\}

Binds an image resource to a named compute shader image uniform entry. 

The binding is stored internally and later applied by \doxylink{class_compute_material_a3c8944e037369072b1685f9e907bc7dc}{Compute\+Material\+::\+Send\+Data()} through \doxylink{class_texture_a28b89a5f6f78de411c85fc83c04963b9}{Texture\+::\+Bind\+As\+Image()}.

If the uniform name is "{}u\+\_\+\+Dst"{}, the texture is additionally stored as the destination texture used by \doxylink{class_render_manager_ae6719dd9b794d26cf923b27f183996dd}{Render\+Manager\+::\+Dispatch\+Compute()} to determine dispatch dimensions.

Reassigning the same uniform name replaces the previous binding.
