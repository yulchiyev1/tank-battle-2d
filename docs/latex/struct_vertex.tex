\doxysubsection{Detailed Description}
Represents a single mesh vertex for the engine\textquotesingle{}s default 2D/quad-\/style pipeline. 

Each vertex currently stores\+:
\begin{DoxyItemize}
\item a 3D position used by the mesh buffer and shader input
\item a 2D UV coordinate used for texture sampling
\end{DoxyItemize}

Although this engine is primarily using the structure for 2D rendering at the moment, the position is stored as glm\+::vec3, which makes the mesh layout compatible with depth-\/aware rendering and future expansion.

\begin{DoxyNote}{Note}
The vertex layout is bound in \doxylink{class_mesh_a6f0136c650ed60f43587e3d8d0c121aa}{Mesh\+::\+Setup\+Mesh()} and \doxylink{class_mesh_a66fa02b2a66182031bb00a9aec134526}{Mesh\+::\+Setup\+Instance\+Attributes()} using attribute location 0 for position and attribute location 1 for UV. \+:content\+Reference\mbox{[}oaicite\+:3\mbox{]}\{index=3\} 
\end{DoxyNote}
