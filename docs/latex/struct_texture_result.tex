\doxysubsection{Detailed Description}
Stores the completed result of an asynchronous texture load. 

\doxylink{struct_texture_result}{Texture\+Result} contains decoded image information produced by the worker thread, including dimensions, channel count, pixel bytes, and the original texture settings.

If loading succeeds, ok is set to true and pixels contains the raw image data. If loading fails, ok is false and the pixel buffer may remain empty. \+:content\+Reference\mbox{[}oaicite\+:7\mbox{]}\{index=7\}
