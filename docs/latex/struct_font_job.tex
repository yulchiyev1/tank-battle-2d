\doxysubsection{Detailed Description}
Describes a font loading request. 

\doxylink{struct_font_job}{Font\+Job} stores the data needed to register a font later, including the font tag, the TTF file path, and the target pixel size.

In the current implementation, the worker thread does not parse or rasterize the font file. It only forwards the metadata into a \doxylink{struct_font_result}{Font\+Result} so the actual font creation can occur in a later stage on the main thread. \+:content\+Reference\mbox{[}oaicite\+:3\mbox{]}\{index=3\}
