\doxysubsection{Detailed Description}
Stores the font reference and UTF-\/8 text content used by a \doxylink{class_text_object}{Text\+Object}. 

\doxylink{struct_text_instance}{Text\+Instance} is a lightweight data container that groups the minimum information required to generate and refresh a text mesh\+:
\begin{DoxyItemize}
\item the font resource used for glyph lookup and atlas access,
\item the actual text string to render.
\end{DoxyItemize}

The font is stored as a weak\+\_\+ptr so that \doxylink{class_text_object}{Text\+Object} does not force ownership of the font resource. Because of that, \doxylink{class_text_object}{Text\+Object} must lock the weak\+\_\+ptr before using it, and safely handles the case where the font has expired.
