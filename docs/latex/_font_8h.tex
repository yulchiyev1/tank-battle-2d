Horizontal alignment options used when generating text geometry. 

This enum controls how each line of text is positioned horizontally when \doxylink{class_font_a8e5834f79c0c38fbc5041d4b2e31ce27}{Font\+::\+Generate\+Text\+Mesh()} builds a mesh.

\item Left keeps the line start at the local origin.
\item Center offsets the line by half of its width.

Vertical alignment options used when generating text geometry. 

This enum controls the vertical starting position of the generated text block when \doxylink{class_font_a8e5834f79c0c38fbc5041d4b2e31ce27}{Font\+::\+Generate\+Text\+Mesh()} builds multiline text geometry.

\item Top starts from the top-\/aligned baseline region.
\item Middle centers the text block vertically.
