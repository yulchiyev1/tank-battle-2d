\doxysubsection{Detailed Description}
Registry that assigns collision group tags to bit masks. 

Collision categories and masks are represented as 32-\/bit bitfields. When an object requests a collision group by tag, a new bit is assigned if the tag has not been seen before. The registry also supports reverse lookup from bit value to tag string.

\begin{DoxyNote}{Note}
The current implementation supports up to 32 unique groups. 
\end{DoxyNote}
