\doxysubsection{Detailed Description}
Loads resource data asynchronously on a background worker thread. 

\doxylink{class_async_resource_loader}{Async\+Resource\+Loader} provides a queue-\/based asynchronous loading pipeline for resource requests such as textures, shaders, fonts, and sounds.

The general workflow is\+:
\begin{DoxyEnumerate}
\item \doxylink{class_async_resource_loader_a81675496e3c264ebbaace9cc50c628a6}{Start()} launches the worker thread.
\item \doxylink{class_async_resource_loader_a1320ce4977f58e8a69689ae1eead3b09}{Enqueue()} pushes one or more \doxylink{_async_resource_loader_8h_ae1b8cce382c5ee957e16be16b2cded39}{Load\+Request} entries into the request queue.
\item \doxylink{class_async_resource_loader_a6d6735825994934b75a0a17574e2bc30}{Worker()} waits for queued requests, processes them, and pushes completed results into an internal completed list.
\item \doxylink{class_async_resource_loader_a3352680b3061a6abb1970de6cd609eb7}{Fetch\+Completed()} is called from the main thread to retrieve finished results and consume them safely.
\end{DoxyEnumerate}

The class tracks overall loading state using\+:
\begin{DoxyItemize}
\item total\+Count\+: total number of queued jobs,
\item loaded\+Count\+: number of jobs processed by the worker.
\end{DoxyItemize}

Progress is reported as loaded\+Count / total\+Count, and \doxylink{class_async_resource_loader_a7274278fd5afa70521a8dfa262fcc18e}{Has\+Finished()} simply checks whether the processed job count has reached the total queued job count. \+:content\+Reference\mbox{[}oaicite\+:12\mbox{]}\{index=12\}

Thread synchronization is handled using\+:
\begin{DoxyItemize}
\item queue\+Mutex for request queue access,
\item completed\+Mutex for completed result access,
\item condition\+\_\+variable to wake the worker when new jobs arrive or shutdown begins.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
This loader is responsible for background file reading and data preparation. Final GPU upload or engine-\/side registration is expected to happen elsewhere. 
\end{DoxyNote}

void Async\+Resource\+Loader\+::\+Stop (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Stops the worker thread and waits for it to finish. 

If the loader is not running, this function returns immediately. Otherwise, it sets running to false under the queue lock, wakes the worker through the condition variable, and joins the worker thread if joinable.

The worker exits only after it observes that loading has been stopped and the request queue has been drained. \+:content\+Reference\mbox{[}oaicite\+:15\mbox{]}\{index=15\}

\begin{DoxyNote}{Note}
This function is synchronous and blocks until the worker thread exits. 
\end{DoxyNote}

void Async\+Resource\+Loader\+::\+Worker (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [private]}}



Worker thread entry point that processes queued requests. 

\doxylink{class_async_resource_loader_a6d6735825994934b75a0a17574e2bc30}{Worker()} runs in a loop while the loader is active. It waits on the condition variable until either\+:
\begin{DoxyItemize}
\item loading is stopped, or
\item at least one request exists in the queue.
\end{DoxyItemize}

For each dequeued request, the worker dispatches behavior based on the concrete request type using std\+::visit\+:


\begin{DoxyItemize}
\item \doxylink{struct_texture_job}{Texture\+Job}\+: Loads image pixels using stb\+\_\+image, fills \doxylink{struct_texture_result}{Texture\+Result}, and stores the result.
\item \doxylink{struct_shader_job}{Shader\+Job}\+: Reads shader source files as text, fills \doxylink{struct_shader_result}{Shader\+Result}, and stores the result.
\item \doxylink{struct_font_job}{Font\+Job}\+: Copies metadata into \doxylink{struct_font_result}{Font\+Result} and stores the result.
\item \doxylink{struct_sound_job}{Sound\+Job}\+: Copies metadata into \doxylink{struct_sound_result}{Sound\+Result} and stores the result.
\end{DoxyItemize}

After finishing each job, loaded\+Count is incremented. \+:content\+Reference\mbox{[}oaicite\+:18\mbox{]}\{index=18\}

\begin{DoxyNote}{Note}
The current implementation does not yet handle \doxylink{struct_sprite_sheet_job}{Sprite\+Sheet\+Job} explicitly. 
\end{DoxyNote}
