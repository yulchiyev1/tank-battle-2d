\doxysubsection{Detailed Description}
Key type used to group objects into the same instanced rendering batch. 

\doxylink{struct_instance_batch_key}{Instance\+Batch\+Key} identifies whether multiple objects can share the same draw batch during instanced rendering.

The current key is composed of\+:
\begin{DoxyItemize}
\item Mesh\texorpdfstring{$\ast$}{*} \+: geometry resource
\item Material\texorpdfstring{$\ast$}{*} \+: material/shader/texture binding state
\item Sprite\+Sheet\texorpdfstring{$\ast$}{*}\+: optional sprite animation source
\end{DoxyItemize}

Objects that share the same mesh, material, and sprite sheet pointer can be grouped together into one batch entry and rendered more efficiently through instancing.

\begin{DoxyNote}{Note}
This key compares raw pointers only. It does not compare the contents of \doxylink{class_mesh}{Mesh}, \doxylink{class_material}{Material}, or \doxylink{class_sprite_sheet}{Sprite\+Sheet}. 
\end{DoxyNote}
