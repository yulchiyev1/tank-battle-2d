\doxysubsection{Detailed Description}
Performs visibility culling against a 2D camera view. 

\doxylink{class_frustum_culler}{Frustum\+Culler} is a stateless helper used by \doxylink{class_render_manager}{Render\+Manager} before batching. The current implementation checks each object and\+:
\begin{DoxyItemize}
\item skips objects that are not alive
\item skips objects that are not visible
\item always keeps objects that ignore the camera
\item otherwise tests the object\textquotesingle{}s world position and bounding radius through \doxylink{class_camera2_d_ac59db8b285e9e7c9646b6e369cfda825}{Camera2\+D\+::\+Is\+In\+View()}
\end{DoxyItemize}

The output list is cleared before results are appended.

Filters a source object list down to only visible objects for the given camera. 

For each object\+:
\begin{DoxyItemize}
\item dead objects are ignored
\item invisible objects are ignored
\item camera-\/ignoring objects are kept automatically
\item all other objects are tested using camera.\+Is\+In\+View()
\end{DoxyItemize}

The object\textquotesingle{}s world position and approximate bounding radius are used for the visibility test.
