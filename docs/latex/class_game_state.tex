\doxysubsection{Detailed Description}
Base class representing one runtime game state in the engine. 

\doxylink{class_game_state}{Game\+State} defines the high-\/level lifecycle for a playable or transitional state such as a title screen, gameplay scene, pause menu, or loading screen.

Each state owns\+:
\begin{DoxyItemize}
\item an \doxylink{class_object_manager}{Object\+Manager} for object lifecycle, update, rendering submission, and collision
\item a \doxylink{class_camera_manager}{Camera\+Manager} for per-\/state camera storage and active camera selection
\end{DoxyItemize}

A \doxylink{class_game_state}{Game\+State} is not driven directly by user code through its public lifecycle methods. Instead, \doxylink{class_state_manager}{State\+Manager} invokes the internal \doxylink{class_game_state_a7210d9f614b4bc65064ade7edee9f38a}{System\+Load()}, \doxylink{class_game_state_a471d08b2632c8696a1105abf356d11e1}{System\+Init()}, \doxylink{class_game_state_a6264f6b70aa45b3e72b6067061cc2395}{System\+Update()}, \doxylink{class_game_state_aebbf582b359e3fb6c929626dba68f8ae}{System\+Free()}, and \doxylink{class_game_state_a165b6e53f7da556b52a29978fef1b812}{System\+Unload()} functions, which in turn call the overridable protected callbacks in the correct engine-\/defined order. \+:content\+Reference\mbox{[}oaicite\+:4\mbox{]}\{index=4\}

The default behavior already provides useful state functionality\+:
\begin{DoxyItemize}
\item \doxylink{class_game_state_a6d11e92e629e2d4cbba792faeb75a71f}{Update()} updates all objects through \doxylink{class_object_manager}{Object\+Manager}
\item \doxylink{class_game_state_ab3f79ba850091b4853d800b1ff2e2974}{Draw()} submits all objects for rendering through \doxylink{class_object_manager}{Object\+Manager}
\item \doxylink{class_game_state_a6264f6b70aa45b3e72b6067061cc2395}{System\+Update()} performs collision checks and optional collider debug rendering
\end{DoxyItemize}

This allows derived states to override only the parts they need while still keeping the engine lifecycle intact.

\begin{DoxyNote}{Note}
\doxylink{class_game_state}{Game\+State} is intended to be subclassed by user-\/defined states.

The state owns its \doxylink{class_object_manager}{Object\+Manager} and \doxylink{class_camera_manager}{Camera\+Manager} directly, so their lifetime matches the lifetime of the state instance. 
\end{DoxyNote}
