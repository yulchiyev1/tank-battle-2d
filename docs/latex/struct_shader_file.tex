\doxysubsection{Detailed Description}
Describes one shader source file used by a shader loading request. 

\doxylink{struct_shader_file}{Shader\+File} pairs a shader stage with the file path containing the source code for that stage. A \doxylink{struct_shader_job}{Shader\+Job} may contain multiple \doxylink{struct_shader_file}{Shader\+File} entries so that a complete shader program can be assembled from multiple source files.
