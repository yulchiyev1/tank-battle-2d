\doxysubsection{Detailed Description}
Describes a sprite sheet creation request. 

\doxylink{struct_sprite_sheet_job}{Sprite\+Sheet\+Job} stores the tag of the sprite sheet to create, the texture tag to reference, and the frame dimensions used to interpret the texture as a grid.

This type is already included in \doxylink{_async_resource_loader_8h_ae1b8cce382c5ee957e16be16b2cded39}{Load\+Request} so the loader interface can be extended to support asynchronous sprite sheet preparation. However, in the current worker implementation, \doxylink{struct_sprite_sheet_job}{Sprite\+Sheet\+Job} is not processed yet. \+:content\+Reference\mbox{[}oaicite\+:5\mbox{]}\{index=5\}

\begin{DoxyNote}{Note}
Because the current Worker() implementation does not handle this request type, enqueuing this job without adding handling logic would leave it unsupported. 
\end{DoxyNote}
