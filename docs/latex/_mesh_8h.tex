enum class \mbox{\hyperlink{_mesh_8h_a240c1924416689e5700ff0d9f30539d9}{Primitive\+Type}}\hspace{0.3cm}{\ttfamily [strong]}}



Specifies the Open\+GL primitive topology used when drawing a mesh. 

This enum is converted to the corresponding Open\+GL draw mode inside the mesh implementation. The selected value determines how the vertex/index data is interpreted when \doxylink{class_mesh_abf84fe4ef735490769268685087e0e32}{Mesh\+::\+Draw()} or \doxylink{class_mesh_ae2933e498cc249ab798bbf265dace77b}{Mesh\+::\+Draw\+Instanced()} is executed.

The mapping is handled internally by the renderer-\/side implementation and is used by both indexed and non-\/indexed mesh draws.

\begin{DoxyNote}{Note}
The actual Open\+GL enum conversion is performed in \doxylink{_mesh_8cpp}{Mesh.\+cpp}. 
\end{DoxyNote}
