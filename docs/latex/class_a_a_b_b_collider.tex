\doxysubsection{Detailed Description}
Axis-\/aligned bounding box collider implementation. 

\doxylink{class_a_a_b_b_collider}{AABBCollider} stores half extents internally. When transform scaling is enabled, \doxylink{class_a_a_b_b_collider_aafbb031b5f15094c94044f7460b2b0f8}{Get\+Half\+Size()} derives the effective extents from the owner\textquotesingle{}s world scale per axis. Otherwise, the manually stored scaled\+Half\+Size is used. \+:content\+Reference\mbox{[}oaicite\+:11\mbox{]}\{index=11\}

AABB-\/vs-\/\+AABB checks compare center distance against the sum of half extents. AABB-\/vs-\/circle checks clamp the circle center to the AABB bounds and compare against the circle radius.
