Nested unordered map type used to store instanced rendering batches by depth and batch key. 

The outer int key typically represents a depth bucket, z-\/order bucket, or another integer-\/based grouping level chosen by the render pipeline.

The inner unordered\+\_\+map groups objects by \doxylink{struct_instance_batch_key}{Instance\+Batch\+Key}, meaning objects that share the same mesh, material, and sprite sheet are collected into the same vector.

In other words, this structure can be interpreted as\+:


\begin{DoxyItemize}
\item first level\+: integer grouping (such as depth)
\item second level\+: shared instancing resource key
\item value\+: list of \doxylink{class_game_object}{Game\+Object} pointers belonging to that batch
\end{DoxyItemize}

This type is intended to make instanced draw submission easier by pre-\/grouping objects that can be rendered together.
