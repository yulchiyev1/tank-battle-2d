\doxysubsection{Detailed Description}
Stores baked glyph metrics and UV information inside the font atlas. 

Each \doxylink{struct_glyph}{Glyph} entry describes one Unicode codepoint that has been baked into the atlas texture. The stored data is later used to measure text and generate quad geometry.

The current implementation stores\+:
\begin{DoxyItemize}
\item pixel size of the bitmap
\item bearing from the baseline/origin
\item advance value from Free\+Type
\item normalized atlas UV coordinates for the glyph rectangle \+:content\+Reference\mbox{[}oaicite\+:3\mbox{]}\{index=3\} 
\end{DoxyItemize}
