Detailed Description

Stores the completed result of a sound loading request.

SoundResult currently carries sound metadata forward to the main thread, including the resource tag, source file path, and looping setting.

The current worker implementation does not decode the sound data itself, so ok defaults to true unless the logic is extended in the future. :contentReference[oaicite:10]{index=10}
