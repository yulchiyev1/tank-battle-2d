Detailed Description

Stores the font reference and UTF-8 text content used by a TextObject.

TextInstance is a lightweight data container that groups the minimum information required to generate and refresh a text mesh:

  • the font resource used for glyph lookup and atlas access,
  • the actual text string to render.

The font is stored as a weak_ptr so that TextObject does not force ownership of the font resource. Because of that, TextObject must lock the weak_ptr before using it, and safely handles the case where the font has expired.
