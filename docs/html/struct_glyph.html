Detailed Description

Stores baked glyph metrics and UV information inside the font atlas.

Each Glyph entry describes one Unicode codepoint that has been baked into the atlas texture. The stored data is later used to measure text and generate quad geometry.

The current implementation stores:

  • pixel size of the bitmap
  • bearing from the baseline/origin
  • advance value from FreeType
  • normalized atlas UV coordinates for the glyph rectangle :contentReference[oaicite:3]{index=3}
