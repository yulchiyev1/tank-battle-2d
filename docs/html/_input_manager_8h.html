enum InputKey

Engine key code aliases matching the GLFW key values used by InputManager.

These constants allow gameplay code to query input without including GLFW directly. The numeric values mirror the GLFW key codes expected by OnKey() and the internal bitset layout. :contentReference[oaicite:35]{index=35}
