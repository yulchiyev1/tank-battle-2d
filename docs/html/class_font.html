Detailed Description

FreeType-backed font resource that manages glyph baking, atlas storage, and text mesh generation.

Font is responsible for:

  • loading a TTF face through FreeType
  • creating and maintaining a single glyph atlas texture
  • lazily baking glyphs on demand
  • providing a text rendering material bound to the atlas
  • measuring UTF-8 text
  • generating a Mesh for aligned text rendering

The current implementation uses a dynamically growing single atlas. Glyphs are not pre-generated for all characters. Instead, characters are baked as needed through TryBakeGlyph(), and the atlas is expanded when there is not enough space. :contentReference[oaicite:5]{index=5}

The material created by this class uses the internal engine text shader registered by RenderManager during initialization, and binds the atlas texture to the uniform "u_FontTexture".

Note
This class owns the FreeType library handle and font face and releases them in the destructor.
atlasVersion is incremented whenever the atlas is rebuilt during expansion. External systems such as TextObject can use this value to detect when text meshes need to be regenerated.
