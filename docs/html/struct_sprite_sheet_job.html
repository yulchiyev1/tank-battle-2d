Detailed Description

Describes a sprite sheet creation request.

SpriteSheetJob stores the tag of the sprite sheet to create, the texture tag to reference, and the frame dimensions used to interpret the texture as a grid.

This type is already included in LoadRequest so the loader interface can be extended to support asynchronous sprite sheet preparation. However, in the current worker implementation, SpriteSheetJob is not processed yet. :contentReference[oaicite:5]{index=5}

Note
Because the current Worker() implementation does not handle this request type, enqueuing this job without adding handling logic would leave it unsupported.
